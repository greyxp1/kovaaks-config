## Installation

Base path for KovaaK:
- Windows: ``C:\Program Files (x86)\Steam\steamapps\common\FPSAimTrainer\FPSAimTrainer``
- Linux: ``~/.steam/steam/steamapps/common/FPSAimTrainer``

| File/Folder | Where to put it |
|---|---|
| `crosshairs/` | `FPSAimTrainer\crosshairs\` |
| `sounds/` | `FPSAimTrainer\sounds\` |
| `Themes/*.json` | `FPSAimTrainer\Saved\SaveGames\Themes\` |
| `UI.json` | `FPSAimTrainer\Saved\SaveGames\` |
| `PrimaryUserSettings.json` | `FPSAimTrainer\Saved\SaveGames\` |
